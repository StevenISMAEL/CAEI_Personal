<h1>proyectoArqui</h1>

## Instalación

1. Clonar repositorio.
   ```sh
   git clone https://github.com/Tatiana-Quilca/proyectoArqui.git proyectoArqui 
   cd Arquiproyect
   code .

2. Backend
    ```sh
    composer install
    cp .env.example .env
    php artisan key:generate
    php artisan migrate --seed
    php artisan serve

3. Frontend
    ```sh
    npm install
    npm run dev
